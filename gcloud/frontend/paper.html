QuickSand White Paper

The QuickSand Framework

QuickSand is a Python Module and command line tool that decodes and decompresses streams within Office Documents or PDF files. Once streams are decoded, it uses Yara to scan for exploits and executables. We provide signatures for document exploits, PDF Exploits and executables through an annual maintenance subscription.

Active Content Detection

QuickSand detects active content that could be used to access information from your computer or download malicious content.

Detect:

  • Javascript
  • Flash
  • Visual Basic
  • ActiveX
  • Macros
  • Shell commands
  • Remotely included templates

Exploit Detection

QuickSand detects known exploits and the active content used to setup memory for exploitation. Identified by CVE, knowing which exploit your malicious document contains can be useful to know what version of Office or PDF Reader is needed to run the exploit in a Sandbox.

Document Similarity Analysis

Our similarity hashes can help link attacks to actors and exploit kits.

  • A hash structhash can link similar documents created with the same tool.
  • The struzzy fuzzy structural hash can be used to calculate a Levenshtein distance between documents. Small distances can indicate an extra exploit or payload was added or removed.

MITRE ATT&CK Integration
